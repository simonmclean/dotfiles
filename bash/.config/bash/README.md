# Bash config

Install these tools for the best experience

- [starship](https://starship.rs) (fancy shell prompt)
- [bat](https://github.com/sharkdp/bat) (better `cat`)
- [eza](https://eza.rocks/) (better `ls`)
- [zoxcide](https://github.com/ajeetdsouza/zoxide) (better `cd`)
- [carapace](https://carapace.sh/) (fancy autocompletion)
- [fzf](https://junegunn.github.io/fzf/) (command line fuzzy finder)
- [ripgrep](https://github.com/BurntSushi/ripgrep) (recursive search that respects gitignore)
